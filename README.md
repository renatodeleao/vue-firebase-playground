

## Project setup
```
cd app
yarn install
```

### Compiles and hot-reloads for development
```
(in app/)
yarn run serve
```

### Compiles and minifies for production
```
(in app/)
yarn run build
```

### Lints and fixes files
```
(in app/)
yarn run lint
```

### Run your unit tests
```
(in app/)
yarn run test:unit
```
